This is a super-duper minimal implementation of a kiosk that integrates with LSApp.
Author: Allan Haggett <allan.haggett@gov.bc.ca>

TO UPDATE:
1. Pull USB drive out of the Raspberry Pi
2. Plug USB drive into your computer
3. Go to the export page in LSApp (https://gww.bcpublicservice.gov.bc.ca/lsapp/export.php)
4. Right-click on the "Classes" and "Save link as"
5. Save that file onto the USB drive, replacing the old "classes.csv"
6. Unplug the USB drive from your computer
7. Plug the USB drive back into the Raspberry Pi
8. Hit F5 on the Raspberry Pi keyboard to refresh the page

Course description

Five self-directed modules in Moodle (approx 2 hrs each) with a live webinar for each (approx 2hrs each): Starting a Project, Planning Project Work, Engaging Interested Parties, Managing Risk and Change & Completing a Project.

Abstract

Applied Project Management 200 focuses on the essential skills needed to perform on a project team in the BC Public Service. In this course, you'll build the knowledge and skills to:

  • Select appropriate tools to ease project planning and execution
  • Identify tasks required to complete project deliverables
  • Prepare appropriate project documentation throughout the project
  • Participate dynamically in a project as a project team member
  • Identify and analyze and monitor risk using a risk register
  • Identify and analyze the impact of change to scope, time, cost and resources

The new Applied learning provides a high-level consideration of different PM approaches which include the spectrum from waterfall to agile. Change management and benefits realization are two other skills that are highlighted as necessary for the applied level learner in Project Management. All modules include realistic case studies from the BC Public Service context across a range of sectors.

In addition, live webinars are available to supplement the online course. Practice performing on a project team in these 2-2.5 hour sessions (one per module).
